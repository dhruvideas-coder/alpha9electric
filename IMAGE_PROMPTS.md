## HOW TO USE

1. Copy the prompt from the gray code block under each image
2. Paste into **Gemini** (gemini.google.com → click image icon)
3. Download the result
4. **Rename it exactly** as shown in the File Name
5. Drop all images into the `images/` folder
6. Done — the website will show them automatically

## TIPS

- Add **"4K, photorealistic, professional photography"** to end of any prompt
- Generate 2-3 times and pick the best result
- For Tennessee city images, try searching Google Images for reference first
- For technical images (panels, transformers), describe what you see in real photos
